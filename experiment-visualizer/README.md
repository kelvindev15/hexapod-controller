# Experiment Visualizer (Vue)

This Vue app visualizes experiment runs generated by the hexapod project.

## What it shows

- Run list across multiple experiment folders
- Summary metrics (status, duration, iterations, model)
- Event type distribution
- Action command distribution
- Full timeline table of parsed `events.jsonl`

## Expected file layout

Pick a parent folder that contains run directories like:

```text
experiments/
	20260410_080340_4c575d4d/
		summary.json
		events.jsonl
	20260410_081500_abcd1234/
		summary.json
		events.jsonl
```

## Run locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite (usually `http://localhost:5173`), then click **Choose Experiments Folder**.

By default, the app also auto-loads runs from the repository `experiments` folder (`../experiments` from this app directory) when running in dev mode.

## Build

```bash
npm run build
```
